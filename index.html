Для чего это

Быстрый рабочий контур

Не декоративная витрина, а рабочая среда для повседневных задач. Здесь можно открыть действующие карты, сделать новый черновик и дальше превратить его в отдельную HTML-страницу для GitHub или выгрузить DOCX для быстрой правки.

Как устроено

Боевые карты и локальные черновики

Опубликованные RoadMap живут как отдельные страницы, а новые идеи и заготовки сначала хранятся локально в браузере. Когда черновик готов, его можно открыть в редакторе и сохранить как отдельный HTML.

Недельная доска отчетов

Управленческий недельный срез по продуктовым направлениям, задачам, контрольным точкам и часам. Подходит для быстрой рассылки, скриншотов и подготовки следующей недели на основе текущей.

  • верхняя матрица продуктовых направлений
  • детализация по проектам и ответственным
  • создание новой недели с очисткой блока «Сделано»

Черновики

Здесь будут появляться новые локальные дорожные карты. Они живут в браузере, не требуют ручного создания файла на старте и подходят для подготовки структуры перед публикацией.

Новый недельный отчет

Создадим локальную копию доски отчетов для новой недели. Все проекты, направления, часы, статусы, дорожные карты и контрольные точки сохранятся, а блок «Сделано» будет очищен для нового заполнения.

При первом открытии нового отчета редактор сам возьмет актуальный шаблон и очистит только недельные тексты в колонке «Сделано».
