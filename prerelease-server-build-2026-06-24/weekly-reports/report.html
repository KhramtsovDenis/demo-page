Заголовок страницы

Направление

Редактируется одно направление. Чтобы изменить существующее, дважды кликните по строке направления в верхней матрице. Кнопка в шапке открывает пустой бланк нового направления.

ID нужен для связей с задачами. Для нового направления он создается автоматически при сохранении. У существующего направления ID не меняем, чтобы не потерять привязки.
Формат: дата | продуктовый результат | что меняется для пользователя. Если оставить пустым, блок продолжит собираться автоматически из release packages.
