# Preview build

Эта папка собрана как безопасная вторичная версия проекта с новой архитектурой отчетов.

Актуально на 2026-06-25:

- это предрелизная сборка, а не замена старого рабочего проекта;
- старую главную страницу на сервере пока не удалять;
- `demo-page-main` не является частью этой сборки и не должен редактироваться;
- неделя 25 открывается через `weekly-reports/report.html?data=week-25_2026-06-18.json`;
- данные недели 25 лежат в `weekly-reports/data/week-25_2026-06-18.json`;
- статус отчета показывается на главной странице, а не внутри самого отчета;
- локальные изменения опубликованного отчета сохраняются в браузере и после обновления страницы должны оставаться на месте.

Как проверять на GitHub Pages:

1. Залить папку `prerelease-server-build-2026-06-24` в корень репозитория рядом со старым `index.html`.
2. Открыть страницу вида `https://<user>.github.io/<repo>/prerelease-server-build-2026-06-24/`.
3. Проверить главную, дорожные карты, недельные отчеты и экспорт.
4. Старую главную страницу не удалять, пока preview не пройдет проверку.

Внутри лежит свой `index.html`, поэтому эта папка работает как отдельный мини-сайт.

Минимальная ручная проверка:

1. Открыть главную.
2. Открыть отчет недели 25.
3. Внести маленькую правку.
4. Нажать сохранение в отчете.
5. Обновить страницу.
6. Убедиться, что правка осталась.
7. Вернуться на главную и проверить, что карточка показывает `Есть изменения`.
